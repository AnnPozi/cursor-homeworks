- git config --global user.name blabla - обьявить свое Имя глобально
- git config --global user.email blabal@bla - обьявить свой имейл глобально
- git init - создание нового Git репозитория
- git status - проверить состояние работы и изменения
- git add - добавление нового или изменного файла
- git commit - сохранение изменение в локальный репозиторий
- git log - отобразить изменение коммита или фильтрация коммитов
- git checkout - обновление файлов в рабочем дереве и навигация между созданными ветками
- git branch - работа с ветками
- git push - подгрузка из локального репозитория на удаленный
- git pull - вытянуть данные из удаленного репозитория в локальный
- git merge - внедрение изменений из другой ветки
- git clone - создание клона(копии) существующего репозитория
- git revert - отмена изменений
- git reset - возврат к предидущей версии
- it rebase - обьеденение изменений двух веток
- git cherry-pick - выбор коммита из ветки и использование его в другой ветке
- git stash - сохранение незакомиченных изменений локально
